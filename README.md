# PokeMatchup

Pokemon Matchup Analysis App

## Overview
This is a Pokemon team builder and matchup analyzer that helps you create optimal teams and analyze type coverage against opponents.

## Features

### 1. Build Your Team
- Search for any Pokemon in the searchbar
- Add Pokemon to your team
- Customize each Pokemon:
  - Adjust levels
  - Select abilities
  - Choose moves from the database

### 2. Select Opponent
- Choose from trainers across different games and generations
- View opponent's team details:
  - Pokemon species
  - Levels
  - Moves
  - Type matchups
  - Abilities

### 3. Matchup Analysis
- Calculate advantage scores based on type matchups
- View your team's strengths and weaknesses
- Identify type coverage gaps
- Get recommendations for types to add for better team coverage
